# Repositório de Atividades de Sistemas Operacionais C++
Este repositório contém atividades desenvolvidas durante o curso de Sistemas Operacionais utilizando a linguagem de programação C. As atividades abordam diversos tópicos relacionados a sistemas operacionais, como gerenciamento de processos, threads, sincronização e comunicação entre processos.

## Conteúdo
Neste repositório, você encontrará exemplos e soluções de problemas relacionados aos seguintes tópicos:
- Processos e threads
- Sincronização de processos e threads (mutexes, semáforos, etc.)
- Comunicação entre processos (pipes, memória compartilhada, etc.)
- Gerenciamento de memória
- Gerenciamento de arquivos e dispositivos de E/S
- Gerenciamento de recursos e escalonamento

## Dependências
Para compilar e executar os exemplos deste repositório, certifique-se de ter instalado em seu sistema:
- Um compilador C, como GCC ou Clang
- Bibliotecas de desenvolvimento do sistema operacional, como pthread para POSIX
